# Sample Resume Website

A clean and responsive HTML/CSS resume website showcasing personal details, skills, work experience, projects, education, and contact information. Designed to provide a professional online presence and easy accessibility for recruiters or collaborators.

## Features

- **Personal Information:** Name, contact details, and social links.  
- **Skills Section:** Highlighting technical and professional skills.  
- **Work Experience:** Details of past roles and responsibilities.  
- **Projects:** Key projects with brief descriptions.  
- **Education:** Academic background and achievements.  
- **Responsive Design:** Fully functional on desktop, tablet, and mobile devices.
